# 🎬 CineSage AI

An AI-powered Movie Information Extractor built using Mistral AI, LangChain, Pydantic, and Streamlit.

## Features

- Movie Information Extraction
- Structured JSON Output
- AI Summary
- Modern Streamlit UI
- Pydantic Validation
